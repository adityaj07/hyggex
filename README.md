## Project Setup

1. _Clone the repository_

```bash
git clone https://github.com/adityaj07/hyggex.git
```

2. _Navigate to the project directory and install dependencies for the project_

```bash
npm install
```

3. _Start the development server_

```bash
npm run dev
```
